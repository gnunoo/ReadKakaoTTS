# 프로젝트 소개

-생활 속에서 사용자가 운전을 하는 등 스마트폰을 직접 보지 못하는 상황이 있다.이런 상황에서 해당 앱에서 카카오톡 메세지를 확인해 TTS 엔진을 통해 메세지를 음성으로 재생해준다. 또는 블루투스 기기가 연결 되어 있을 시 기기를 통해 소리가 재생 된다.스마트폰을 만질 수 없는 상황이거나 볼 수 없는 상황에서 메세지를 읽어줌으로써 사용자에게 편리함을 제공한다.
